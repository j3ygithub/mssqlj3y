## Installation

OS X & Linux:

```
python3 -m venv /Users/jimmy_lin/envs/mssqlj3y
source /Users/jimmy_lin/envs/mssqlj3y/bin/activate
git clone https://github.com/j3ygithub/mssqlj3y /Users/jimmy_lin/repos/mssqlj3y
pip install -r /Users/jimmy_lin/repos/mssqlj3y/requirements.txt
python /Users/jimmy_lin/repos/mssqlj3y/manage.py runserver
```

Windows:

```
python -m venv C:\Users\jimmy_lin\envs\mssqlj3y
C:\Users\jimmy_lin\envs\mssqlj3y\Scripts\activate
git clone https://github.com/j3ygithub/mssqlj3y C:\Users\jimmy_lin\repos\mssqlj3y
pip install -r C:\Users\jimmy_lin\repos\mssqlj3y\requirements.txt
python C:\Users\jimmy_lin\repos\mssqlj3y\manage.py runserver
```
